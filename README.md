1. You provide **info.md**
2. Codex generates `instructions.md`
3. Codex generates sorted, numbered `tasks.md`
4. You say:
   ```
   Proceed with Task 1
   ```
5. Codex:
   - Executes the task
   - Modifies files
   - Creates specs
   - Builds UI or backend
   - Generates a history file
   - Marks task complete
6. You say:
   ```
   Proceed with Task 2
   ```
7. Codex repeats the cycle until the entire project is complete.

This creates a **living, self-evolving AI textbook engine**.
